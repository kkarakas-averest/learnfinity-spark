How It Works

1. Data Collection & Preparation

We start with structured data about the employee's skills and the target role requirements. This data is organized using a consistent taxonomy of financial skills.

2. AI-Powered Course Design

The system analyzes skill gaps and generates a personalized course outline using the Groq API and llama-3.3-70b-versatile model. The AI creates a progressive learning journey that builds on existing skills.

3. Content Generation

For each module, the AI generates highly detailed, personalized content that includes explanations, examples, and connections to the learner's existing skills. This is all accomplished through strategic prompting.
